### 2. GSAP Animation Engine (`gsap-*`)

Agaate relies heavily on GSAP 3.15 and `@tw-animate-css` for scroll-triggered storytelling and smooth micro-animations.

- **[gsap-core](file:///c:/Users/krish/Downloads/agaate/.agents/skills/gsap-core/SKILL.md)**: Core API guidance (`gsap.to()`, `from()`, `fromTo()`, easing, `gsap.matchMedia()` for responsive and prefers-reduced-motion).
- **[gsap-react](file:///c:/Users/krish/Downloads/agaate/.agents/skills/gsap-react/SKILL.md)**: Best practices for GSAP in React/TanStack Start (`useGSAP()` hook, ref management, `gsap.context()`, and automatic cleanup).
- **[gsap-scrolltrigger](file:///c:/Users/krish/Downloads/agaate/.agents/skills/gsap-scrolltrigger/SKILL.md)**: Building scroll-linked animations, pinning sections, and scrubbing parameters (`ScrollTrigger`).
- **[gsap-timeline](file:///c:/Users/krish/Downloads/agaate/.agents/skills/gsap-timeline/SKILL.md)**: Sequencing keyframes and complex choreographies (`gsap.timeline()`, position parameters, nesting).
- **[gsap-performance](file:///c:/Users/krish/Downloads/agaate/.agents/skills/gsap-performance/SKILL.md)**: Optimizing for 60fps (`will-change`, transform preference, avoiding layout thrashing, batching).
- **[gsap-plugins](file:///c:/Users/krish/Downloads/agaate/.agents/skills/gsap-plugins/SKILL.md)**: Official GSAP plugins (`SplitText`, `Observer`, `Flip`, `ScrollSmoother`, `Draggable`, `CustomEase`).
- **[gsap-utils](file:///c:/Users/krish/Downloads/agaate/.agents/skills/gsap-utils/SKILL.md)**: Utility functions (`gsap.utils.clamp`, `mapRange`, `interpolate`, `toArray`, `snap`).
